# SINERGIA - Plataforma de Automatización e IA

Este proyecto consiste en una Landing Page de alta gama con un backend potenciado por IA para la calificación de leads.

## Estructura del Proyecto

- `/` (Raíz): Contiene el Frontend (HTML, CSS, JS e Imágenes).
- `/backend`: Contiene el servidor Node.js (Express, Gemini AI, Supabase SDK).

## Configuración del Backend

1. Entra a la carpeta `backend/`.
2. Instala las dependencias: `npm install`.
3. Crea un archivo `.env` basado en los requisitos del código.
4. Inicia el servidor: `npm run dev`.

## Despliegue en Coolify

- **Frontend**: Configurar como un sitio estático apuntando a la raíz del repositorio.
- **Backend**: Configurar como una aplicación Node.js apuntando a la subcarpeta `/backend`.
